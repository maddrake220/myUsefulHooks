# hook components 
  - useDeviceOrientation 
    기기의 기울임, 방향에 따라 alpha, beta, gamma 값을 가져온다.
  - useFavicon
    Favicon 설정
  - useGeolocation
   현재 위치 gps값 계산
  - useKeyPress
  - 특정 키 입력시 값을 get
  - useLocalStorage
    LocalStorage에 key,value값 저장 관련
  - useLockScroll
    스크롤바 제어 
  - useMousePosition
    현재 마우스 위치 값 get
  - useOnlineStatus
    Network환경 값
